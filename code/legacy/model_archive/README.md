# Model Archive

`t2dyn_evolutionary_model/` is the staged T2DYN model tree copied from `/Users/dlev2617/Documents/~~Archives/MNCPPC/t2dyn`.

The archive includes annual model inputs and outputs, run scripts, Travel/2/EMME-oriented files, and legacy compressed `.Z` originals. `SOURCE_TREE_MANIFEST.csv` records SHA-256 checksums and coarse role labels for each retained file.

The archive is preserved for reproducibility provenance. It should not be described as a modern standalone reproduction script: it depends on historical HP-UX-era paths and EMME/2/Travel/2 tooling.
